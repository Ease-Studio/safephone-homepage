Hướng Dẫn Tạo API Key OpenAI và DeepSeek

Các mô hình AI hiện đại như OpenAI và DeepSeek cho phép nhà phát triển tích hợp trí tuệ nhân tạo vào ứng dụng, website hoặc hệ thống tự động hóa thông qua API. Để sử dụng các dịch vụ này, bạn cần tạo một API Key, đây là mã xác thực giúp ứng dụng của bạn giao tiếp với hệ thống AI.

Trong bài viết này, chúng tôi sẽ hướng dẫn từng bước cách tạo API Key từ OpenAI và DeepSeek.

Tạo API Key OpenAI

Trang chính thức của OpenAI:

https://platform.openai.com/

Bước 1: Đăng ký hoặc đăng nhập tài khoản OpenAI

Truy cập nền tảng dành cho nhà phát triển của OpenAI tại:

https://platform.openai.com/

Đăng nhập hoặc tạo tài khoản mới nếu bạn chưa có tài khoản OpenAI.

Hình 1: Trang đăng nhập OpenAI Platform.

Bước 2: Truy cập mục API Keys

  1. Nhấn vào ảnh đại diện tài khoản.
  2. Chọn View API Keys hoặc truy cập trực tiếp:

https://platform.openai.com/api-keys

Hình 2: Trang quản lý API Keys của OpenAI.

Bước 3: Tạo API Key mới

  • Nhấn nút Create new secret key.
  • Đặt tên cho API Key.
  • Xác nhận tạo khóa.

Hình 3: Tạo API Key mới trên OpenAI.

Sau khi tạo thành công, hệ thống sẽ hiển thị API Key duy nhất một lần. Hãy lưu trữ cẩn thận vì bạn sẽ không thể xem lại toàn bộ khóa sau đó.

Bước 4: Thiết lập thanh toán

Để sử dụng API, bạn cần có phương thức thanh toán hợp lệ hoặc số dư trong tài khoản.

Trang Billing:

https://platform.openai.com/settings/organization/billing

Hình 4: Thiết lập thanh toán và quản lý chi phí API.

Tạo API Key DeepSeek

Trang chính thức của DeepSeek:

https://platform.deepseek.com/

Bước 1: Đăng ký hoặc đăng nhập DeepSeek Platform

Truy cập:

https://platform.deepseek.com/

Đăng nhập hoặc tạo tài khoản DeepSeek.

Hình 5: Trang đăng nhập DeepSeek Platform.

Bước 2: Truy cập API Keys

  1. Mở Dashboard.
  2. Chọn mục API Keys.
  3. Nhấn Create API Key.

Hình 6: Trang quản lý API Keys của DeepSeek.

Bước 3: Tạo API Key mới

  • Đặt tên cho API Key.
  • Xác nhận tạo khóa.
  • Sao chép khóa vừa được tạo.

Hình 7: Tạo API Key trên DeepSeek.

DeepSeek cũng chỉ hiển thị khóa đầy đủ một lần sau khi tạo thành công.

Bước 4: Nạp số dư sử dụng API

DeepSeek hoạt động theo mô hình tính phí dựa trên lượng token sử dụng.

  • Mở mục Wallet hoặc Billing.
  • Nạp số dư vào tài khoản.
  • Kiểm tra lượng token tiêu thụ trong Dashboard.

Hình 8: Nạp tiền và theo dõi chi phí API trên DeepSeek.

Lưu Ý Bảo Mật API Key

API Key tương tự như mật khẩu của dịch vụ AI. Nếu bị lộ, người khác có thể sử dụng tài khoản của bạn và phát sinh chi phí ngoài ý muốn.

  • Không đăng API Key lên GitHub công khai.
  • Không nhúng API Key trực tiếp trong ứng dụng client.
  • Lưu trữ bằng Environment Variables hoặc Secret Manager.
  • Thu hồi và tạo khóa mới nếu nghi ngờ bị lộ.

Sử Dụng API Key Trong Ứng Dụng

Sau khi tạo thành công API Key, bạn có thể sử dụng AI Key này để nhập vào ô API Key trong SafePhone. Các bước thực hiện như sau:

  • Mở ứng dụng SafePhone
  • Bấm vào nút có chữ "AI" ngay bên cạnh tiêu đề "SafePhone", 1 hộp thoại sẽ hiện lên
  • Bấm kích hoạt AI
  • Chọn nhà cung cấp AI: OpenAI hoặc DeepSeek
  • Dán API Key vào ô "API Key"
  • Bấm "Lưu lại"
  • Sau khi hoàn tất, nút có chữ "AI" sẽ chuyển sang màu xanh
