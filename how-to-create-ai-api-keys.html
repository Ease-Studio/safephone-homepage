Hướng Dẫn Tạo API Key OpenAI và DeepSeek

Tăng cường bảo mật trên điện thoại di động

Nếu bạn đang tìm kiếm giải pháp để phòng tránh các website giả mạo hoặc website phát tán mã độc thì hãy tải ngay ứng dụng SafePhone!

Tải SafePhone từ PlayStore

Các mô hình AI hiện đại như OpenAI và DeepSeek cho phép nhà phát triển tích hợp trí tuệ nhân tạo vào ứng dụng, website hoặc hệ thống tự động hóa thông qua API. Để sử dụng các dịch vụ này, bạn cần tạo một API Key, đây là mã xác thực giúp ứng dụng của bạn giao tiếp với hệ thống AI.

Trong bài viết này, chúng tôi sẽ hướng dẫn từng bước cách tạo API Key từ OpenAI và DeepSeek.

Tạo API Key OpenAI

Trang chính thức của OpenAI: https://platform.openai.com/

Bước 1: Đăng ký hoặc đăng nhập tài khoản OpenAI

Truy cập nền tảng dành cho nhà phát triển của OpenAI tại: https://platform.openai.com/

Đăng nhập hoặc tạo tài khoản mới nếu bạn chưa có tài khoản OpenAI.

Bước 2: Truy cập mục API Keys

  1. Bấm vào mục "API keys" (trên thanh menu bên trái màn hình)
  2. Bấm vào + Create new secret key, một hộp thoại sẽ hiện lên

Bước 3: Tạo API Key mới

  • Trên hộp thoại mới hiện lên, đặt tên cho API key ở ô "Name" (bất kì tên gì cũng được, ví dụ: SafePhone AI)
  • Bấm vào nút "Create secret key" để tạo khoá, một hộp thoại khác sẽ hiện lên
  • Trên hộp thoại này hiển thị API key bạn cần lưu trữ. Hãy sao chép API key đó và lưu vào đâu đó để có thể copy & paste vào ứng dụng SafePhone. (Lưu ý: API key tương tự như mật khẩu, không được để lộ API key với người khác.

Bước 4: Thiết lập thanh toán

Để sử dụng API Key của OpenAI, bạn cần có phương thức thanh toán hợp lệ hoặc số dư trong tài khoản. Để nạp tiền cho API key trên OpenAI, thực hiện theo các bước sau:

  • Truy cập trang Billing: https://platform.openai.com/settings/organization/billing/overview
  • Tham khảo bảng giá của OpenAI tại: https://openai.com/api/pricing/
  • Bấm vào "Add payment details", điền thông tin thanh toán và thực hiện chuyển khoản như thông thường.

Tạo API Key DeepSeek

Trang chính thức của DeepSeek: https://platform.deepseek.com/

Bước 1: Đăng ký hoặc đăng nhập tài khoản DeepSeek

Truy cập nền tảng dành cho nhà phát triển của DeepSeek tại: https://platform.deepseek.com/

Đăng nhập hoặc tạo tài khoản mới nếu bạn chưa có tài khoản DeepSeek.

Bước 2: Truy cập mục API Keys

  1. Bấm vào mục "API keys" (trên thanh menu bên trái màn hình)
  2. Bấm vào Create new API key, một hộp thoại sẽ hiện lên

Bước 3: Tạo API Key mới

  • Trên hộp thoại mới hiện lên, đặt tên cho API key ở ô "Name" (bất kì tên gì cũng được, ví dụ: SafePhone AI)
  • Bấm vào nút "Create API key" để tạo khoá, một hộp thoại khác sẽ hiện lên
  • Trên hộp thoại này hiển thị API key bạn cần lưu trữ. Hãy sao chép API key đó và lưu vào đâu đó để có thể copy & paste vào ứng dụng SafePhone.(Lưu ý: API key tương tự như mật khẩu, không được để lộ API key với người khác.)

Bước 4: Thiết lập thanh toán

Để sử dụng API key của DeepSeek, bạn cần có phương thức thanh toán hợp lệ hoặc số dư trong tài khoản. Để nạp tiền cho API key trên DeepSeek, thực hiện theo các bước sau:

  • Truy cập trang Top-up của DeepSeek : https://platform.deepseek.com/top_up
  • Tham khảo bảng giá của OpenAI tại: https://api-docs.deepseek.com/quick_start/pricing/
  • Điền thông tin thanh toán và thực hiện chuyển khoản như thông thường.

Lưu Ý: Bảo Mật API Key

API Key tương tự như mật khẩu của dịch vụ AI. Nếu bị lộ, người khác có thể sử dụng tài khoản của bạn và phát sinh chi phí ngoài ý muốn. Bạn có thể lưu API Key vừa được tạo theo các cách sau để đảm bảo tính bảo mật:

  • Tạo 1 file Word trên máy tính, copy & paste API vào file Word đó, giấu ở nơi an toàn hoặc đặt mật khẩu cho file Word
  • Tạo 1 file trên Drive của bạn, lưu API Key vào đó và thiết lập chỉ mình bạn mới được xem file đó.
  • Thường xuyên kiểm tra mức độ sử dụng của API Key trên trang chủ của OpenAI hoặc DeepSeek, tương ứng tại: https://platform.openai.com/usage hoặc https://platform.deepseek.com/usage, nếu phát hiện thấy bất thường, hãy xoá API key và tạo mới API key khác.

Sử Dụng API Key Trong Ứng Dụng

Sau khi tạo thành công API Key, bạn có thể sử dụng AI Key này để nhập vào ô API Key trong SafePhone. Các bước thực hiện như sau:

  • Mở ứng dụng SafePhone
  • Bấm vào nút có chữ "AI" ngay bên cạnh tiêu đề "SafePhone", 1 hộp thoại sẽ hiện lên
  • Bấm kích hoạt AI
  • Chọn nhà cung cấp AI: OpenAI hoặc DeepSeek
  • Dán API Key vào ô "API Key"
  • Bấm "Lưu lại"
  • Sau khi hoàn tất, nút có chữ "AI" sẽ chuyển sang màu xanh
